# Image Structure

- `brand/` shared YABI Studio logos
- `icons/projects/` reusable icons used in project detail pages
- `icons/services/` reusable icons used on the services page
- `about/` visuals used on `o-nas.html`
- `projects/<project-slug>/` thumbnails and detail images for each project page
- `articles/<article-slug>/` thumbnails and article visuals grouped by article topic
- `shared/` fallback assets used in multiple places

# Naming Rules

- Filenames use readable SEO slugs instead of Webflow hash prefixes.
- Project assets start with the project slug, for example `devio-website-thumbnail.png`.
- Article assets start with the article topic slug, for example `seo-webflow-illustration-1.jpg`.
- Shared icons describe their purpose, for example `project-client-icon.svg`.
