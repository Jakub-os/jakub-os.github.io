# YABI Nav Preview

This project is a standalone static preview built with plain HTML, CSS, and JavaScript.

## Files

- `index.html` contains the page markup and nav component.
- `styles.css` contains the styling.
- `script.js` handles the menu interaction and keyboard support.
- `hero.html` contains the standalone hero section implemented from Figma.
- `hero.css` contains the hero page styling.
- `images/` contains the image assets used by the menu panel and hero page.

## Running the preview

Open `index.html` for the nav prototype or `hero.html` for the Figma hero preview.
